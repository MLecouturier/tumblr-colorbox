Tumblr-colorbox

Tumblr javascript for images. Creates a lightbox using jquery colorbox

Download this project as a .zip file Download this project as a tar.gz file

Jquery lightbox with colorbox.js inside your tumblr blog


  Add the following javascript to the head section of your tumblr theme:
  
  http://29-steps-uk.github.com/tumblr-colorbox/javascripts/tumblr-colorbox.js

To view it in action please check out our own tumblr blog at blog.29steps.co.uk and click on the photosets and photos posts.
